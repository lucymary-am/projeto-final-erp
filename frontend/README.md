# Orbis - Sistema de Gestão Empresarial

Frontend desenvolvido em Angular 21 com HTML, CSS e TypeScript para o sistema Orbis de gestão empresarial.


### Pré-requisitos
- Node.js (v18 ou superior)
- npm (v9 ou superior)
- Backend MySQL rodando com as credenciais definidas em `.env`

### Instalação do Frontend

```bash
npm install
```

### Iniciar o Backend

No diretório do backend:

```bash
npm install
npm run dev
```

O backend será iniciado em `http://localhost:3001`.

Backend: https://github.com/lucymary-am/projeto-final-erp

### Iniciar o Frontend

No diretório do frontend:

```bash
npm start
```

Abra `http://localhost:4200/` no navegador.
